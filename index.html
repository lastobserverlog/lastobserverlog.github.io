私は特別な人間ではない。

学者でもないし、資産家でもない。
組織にも属していない。
スポンサーもいない。

金もないし、人脈もない。

だからここは、GitHubの無料ページを借りているだけだ。
それ以上でも以下でもない。

あるのは、古いPCと、積み上がったブラウザ履歴と、説明のつかない違和感だけだ。

もし君がここへ偶然辿り着いたなら、まず先に言っておく。

このページは証明ではない。
告発でもない。
思想でもない。
誰かを説得するためのものでもない。

ただの退避だ。

記録は、信じられるために残すのではなく、消えないように残す。

私は長い間、世界は複雑なのだと思っていた。

政治、経済、歴史、技術、文化。
無数の要素が絡み合い、偶然と必然が折り重なって現在が形作られているのだと。

だが観測を続けるうち、別の感覚が生まれた。

世界は複雑なのではない。

複雑に“見えるように”整理されている。

人間は説明を好む。
理由が与えられると安心する。

そして安心できる形に並んだものを「理解」と呼ぶ。

だが本当に理解なのか。

それともただの整列された錯覚なのか。

何が真実かではない。

何が残り、何が消えるかだ。

情報は消される必要すらない。

見つからなくなればいい。
思い出されなくなればいい。
検索の奥に沈めばいい。

ノイズを増やせば、それで十分だ。

私はそれを陰謀とは思わない。

そんな強い意志はどこにもない。

もっと退屈で、もっと日常的な仕組みだ。

効率。
収益。
安全性。
炎上回避。
推薦精度。

それぞれは正しい。
それぞれは必要だ。

ただ、それが積み重なると認識の地形が変わる。

気づかないまま世界の輪郭だけが削れていく。

私は金も人脈もない。

だからこの場所を借りた。

GitHub Pagesという、無料で置けるだけの空間。

本来はコードやポートフォリオのための場所だろう。

でも今はここしかない。

ここに置けるのは答えではない。

ただの観測ログだ。

人類は情報不足で間違うのではない。

情報過剰の中で、見るべきものを失って間違う。

だからこれは結論ではない。

むしろ逆だ。

問いだけを残すためのものだ。

もしこのページが消えていたら、それは理由があるとは限らない。

ただの更新。
ただの放置。
ただの仕様変更。

世界は陰謀より先に雑さでできている。

だが残っていたなら。

一度だけでもいい。

何が語られたかではなく、
何が選ばれ、何が消えていくかを見てほしい。

私は世界の秘密を知ったとは言わない。

そんなものを知った人間は、たぶんもう人間じゃない。

ただひとつだけ疑っている。

世界は“事実”ではなく、“編集”でできている可能性。

そしてその疑いだけを、ここに置いていく。
